# DMC Catalog (Android)

Offline Android app for managing DMC thread colors and personal stash.

## Build APK with GitHub Actions

1. Push this project to a GitHub repository.
2. Open the repository in GitHub.
3. Go to **Actions** -> **Build Android APK**.
4. Click **Run workflow**.
5. When it finishes, open the run and download the artifact `dmc-catalog-debug-apk`.
6. Inside the artifact, use `app-debug.apk`.

No Android SDK or Gradle installation is required on your local machine for this flow.
